# 작업 완료 문서 아카이브

이 폴더는 완료된 작업들의 요약 문서를 보관합니다.

## 폴더 구조

- **BE_P1-5_***: P1-5 작업 시리즈 완료 요약
- **BE_SUB_PROMPT_***: 서브 프롬프트 작업 완료 요약
- **BE_HOTFIX_***: 핫픽스 작업 완료 요약
- **P0_***, **P1_***: 우선순위 작업 완료 요약
- **PR_ROLES_***: Roles 관련 PR 작업 완료 요약
- **BE_REFACTOR_***: 리팩토링 작업 완료 요약

## 참고

이 문서들은 과거 작업 이력을 보관하기 위한 것입니다.
현재 작업은 `docs/standardization/` 또는 `docs/workdone/` (현재는 이 폴더로 통합됨)를 참고하세요.
