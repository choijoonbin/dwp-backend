# 트러블슈팅 문서 아카이브

이 폴더는 해결된 문제들의 트러블슈팅 문서를 보관합니다.

## 주요 문서

- **BYTEA_***: PostgreSQL BYTEA 타입 관련 이슈 해결
- **IDE_***: IDE 설정 및 빌드 경로 관련 이슈 해결
- **MIGRATION_***: Flyway 마이그레이션 관련 이슈 해결
- **MONITORING_***: 모니터링 API 관련 이슈 해결
- **SPRING_BOOT_***: Spring Boot 버전 관련 이슈 해결

## 참고

이 문서들은 과거에 발생했던 문제와 해결 방법을 보관하기 위한 것입니다.
현재 문제 해결은 `docs/guides/` 또는 `docs/essentials/RUNBOOK_BACKEND.md`를 참고하세요.
